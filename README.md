# AI_Chatbot-Internship_task_3
A basic rule-based chatbot built in Python as part of my CodeAlpha Python Programming Internship.

## About
A simple console chatbot that replies to a few predefined greetings and questions. No AI or external APIs — just plain if-elif logic to match keywords in what you type.

## How it works
- Type a message like "hello", "how are you", or "bye".
- The bot checks your message for keywords and replies accordingly.
- Type "bye" anytime to end the conversation.

## Concepts Used
- Functions
- if-elif statements
- Loops
- Input/output handling

## How to run
```
python chatbot.py
```

## Sample Output
```
Chatbot: Hi! Type 'bye' to end the chat.

You: hello
Chatbot: Hi! How can I help you today?
You: how are you
Chatbot: I'm fine, thanks! How about you?
You: bye
Chatbot: Goodbye! Have a great day.
```
